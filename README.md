# Information Storage and Retrieval (ISR) Assignment

## Overview
This repository contains the implementation of ISR course assignments (HW1–HW8).  
The project covers core Information Retrieval concepts including preprocessing, indexing, retrieval models, clustering, topic modeling, and evaluation techniques.
## Project Contents

### HW1
- Text preprocessing
- Tokenization
- Stopword removal

### HW2
- Inverted indexing
- Term frequency processing

### HW3
- Query processing
- Retrieval model implementation

### HW4
- Vector Space Model

### HW5
- Evaluation metrics
  - Precision
  - Recall
  - MAP
  - nDCG

### HW6
- KMeans clustering

### HW7
- LDA Topic Modeling

### HW8
- Final evaluation and improvements

## Technologies Used
- Python 3
- NLTK
- Scikit-learn
- NumPy
- Pandas

## How to Run

1. Clone the repository

git clone https://github.com/yanetdereje35-eng/isr-assignment.git

2. Open the project in VS Code or any Python IDE

## Results
The system successfully:
- Processes documents
- Builds indexes
- Retrieves relevant documents
- Performs clustering
- Extracts topics using LDA
- Evaluates retrieval performance using standard IR metrics
